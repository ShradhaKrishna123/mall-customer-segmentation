# Mall Customer Segmentation using KMeans Clustering

## 📌 Project Overview

This project performs **customer segmentation** using **KMeans clustering** on mall customer data.

The goal is to group customers based on their behavior to help businesses:

* Understand customer segments
* Target marketing strategies
* Improve sales and engagement

---

## 📊 Dataset Features

* Customer ID
* Gender
* Age
* Annual Income
* Spending Score

---

## ⚙️ Workflow

1. Load dataset
2. Data preprocessing
3. Feature selection (Income & Spending Score)
4. Determine optimal clusters using Elbow Method
5. Apply KMeans clustering
6. Visualize customer segments

---

## 📊 Key Concepts

* Unsupervised Learning
* KMeans Clustering
* Elbow Method
* Customer Segmentation
